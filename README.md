# 🎮 Contexto Twitch Overlay — JaskullTV

Overlay per OBS con leaderboard live per il gioco Contexto.

## Come aggiornare la classifica

Modifica il file `leaderboard.txt` direttamente da GitHub:
1. Clicca su `leaderboard.txt` qui sopra
2. Clicca l'icona matita (✏️) in alto a destra
3. Modifica i punteggi
4. Clicca **"Commit changes"**

L'overlay si aggiorna automaticamente ogni 30 secondi!

## Formato del file

```
# Le righe con # sono commenti
NomePlayer|Punteggio
```

## URL per OBS Browser Source

```
https://TUO-USERNAME.github.io/TUO-REPO/
```
